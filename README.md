# Project Management App

A full-featured project management application with Kanban boards, task lists, Gantt timelines, and dashboards.

## Features

- **Dashboard** — Stats cards, pie chart, bar chart, activity feed, upcoming deadlines
- **Kanban Board** — Drag-and-drop columns (To Do, In Progress, Review, Done)
- **Task List** — Searchable, filterable, sortable table
- **Timeline/Gantt** — Visual task timeline with today marker
- **Task Details** — Full modal with subtasks, comments, and editing
- **Create Tasks** — Add new tasks with all fields

## Tech Stack

- React 18
- Recharts (charts)
- Tailwind CSS
- Single-page static site

## Deployment

Deploy to Netlify by connecting this repo. No build step required — it's a static site.
